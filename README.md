# Отчет по проекту: Предсказание цены автомобиля

## 1. Этапы работы

- EDA: анализ распределений, пропусков, дубликатов, корреляций
- Preprocessing: очистка mileage, engine, max_power, torque
- Feature Engineering: power_per_engine, year_squared, km_driven_log
- Обучение: LinearRegression, Lasso, Ridge, ElasticNet
- Оптимизация: GridSearchCV (10 фолдов)
- Деплой: Streamlit-приложение

## 2. Результаты

| Модель | Test R2 |
|--------|---------|
| LinearRegression | 0.5577 |
| Lasso | 0.5585 |
| Ridge | 0.5577 |
| ElasticNet | 0.5516 |
| Log Target | 0.8115 |

Линейные модели объясняют 56% дисперсии цены. Логарифмирование целевой переменной повысило R2 до 0.81.

## 3. Наибольший прирост

Логарифмирование цены (R2 вырос с 0.56 до 0.81). Причина — скошенное распределение целевой переменной.

## 4. Что не удалось

- Учесть категориальные признаки (fuel, seller_type, transmission, owner, brand) — требуется OHE и больше данных
- Достичь R2 > 0.85 — линейная модель не улавливает нелинейные зависимости

## 5. Оценка сервиса

Достоинства: загрузка файлов, ручной ввод, графики EDA, прогноз, визуализация весов.
Недостатки: модель работает только на 4 числовых признаках.
